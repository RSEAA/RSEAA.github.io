# Contributing

## Running the site locally

1. **Install dependencies and serve:**
   ```bash
   bundle install
   bundle exec jekyll serve
   ```
   Then open http://localhost:4000

For more guides, see [Jekyll installation](https://jekyllrb.com/docs/installation/). 

## Structure of the repository 

| Path | Purpose |
|------|--------|
| `_config.yml` | Site config, nav (`header_pages`), logo |
| `_includes/` | Header and footer (navbar, theme toggle) |
| `_layouts/default.html` | Main page layout |
| `assets/css/rse-aunz.css` | Custom styles (theme, sticky header, footer) |
| `index.md` | Homepage |
| `conferences/` | Conference info and past events (RSAA22–RSAA26) |
| `about/` | Accessibility fellowship, volunteering, reports |
| `marketing/` | Marketing and prospectus content |
