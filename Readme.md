# Research Software Asia Australia Conference (RSAA26)

Website for the **Research Software Asia Australia Conference** — a community event for Research Software Engineers (RSEs) across Asia and Australia, run jointly by the [RSE Asia Association](https://rse-asia.github.io/RSE_Asia/) and the [RSE Association of Australia and New Zealand](https://rse-aunz.github.io/).

- **Live site:** [rseaa.org](https://rseaa.org/)
- **RSAA26:** 25–28 August 2026 · theme: *Connecting with Community* · #RSAA26

---

## Repository

This repository contains the source for the RSEAA conference website. It is built with [Jekyll](https://jekyllrb.com/) (theme: [Minima](https://github.com/jekyll/minima)) and can be served with GitHub Pages or locally.
