# Assets

Static assets are organised into:

- **css/** – Stylesheets (e.g. `rse-aunz.css`)
- **images/** – Logos, banners, slides, and other images (PNG, SVG, WebP)
- **calendars/** – ICS calendar files for conference dates
- **documents/** – PDFs and other documents (prospectuses, presentation slides)

References in the site use paths like `/assets/images/rseaa_logo.png`, `/assets/calendars/RSEAA24.ics`, and `/assets/documents/RSEAA24 Prospectus.pdf`.
